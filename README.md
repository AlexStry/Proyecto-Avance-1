# Hub de Juegos de Palabras
Interés: Desde pequeño, una de mis cosas favoritas ha sido jugar videojuegos. No hay nada mejor que, después de un día estresante, un día largo o un día cansado, llegar a tu casa y poder jugar videojuegos con tus amigos en línea. Naturalmente, cuando en nuestra clase de C++ tuvimos que crear un proyecto, decidí crear un hub de videojuegos. No tengo habilidades muy desarrolladas para poder crear un hub de juegos avanzados, así que para este proyecto decidí crear un hub de juegos muy simples, para quizás en el futuro lograr crear uno más complicado.

Contexto: Mi proyecto se llama "STRYER GAME HUB" y es un hub muy simple que tiene solo 3 videojuegos. El primer videojuego es el clásico de piedra, papel o tijeras. Eligiendo una de las opciones estarás compitiendo contra la opción escogida al azar por la computadora, donde: Piedra > Tijeras, Papel > Piedra y Tijeras > Papel. El segundo juego dentro de mi hub de juegos es un juego donde tienes 10 intentos para adivinar un número generado al azar por la computadora, mientras que la computadora te dice si el número que adivinas es mayor o menor al número aleatorio generado. Finalmente, el último juego que implementé dentro de mi hub es el simple "Sol o Águila", donde escoges sol o águila y si la moneda cae en tu decisión, ganas el juego. Para agregar un poco de complejidad a mi proyecto y que se sienta un poco más completo, decidí agregar una forma de subir de nivel. Al iniciar el código, te pedirá poner el nombre de tu usuario (creando un archivo txt si no tienes un usuario ya). Después de entrar con tu usuario, cada juego te dará una cantidad diferente de XP para subir de nivel, dependiendo de la complejidad del juego: Piedra, papel o tijeras da 10, Adivina el número puede dar 90 más, dependiendo de cuántos intentos tomes en adivinar, y sol o águila da 5 si ganas. Al obtener 20 puntos de XP, tu usuario subirá de nivel.
